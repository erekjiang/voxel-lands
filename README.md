# Voxel Lands · 体素大陆

一个用 HTML + Three.js 从零实现的第一人称体素沙盒游戏，浏览器直接可玩。所有贴图、图标、音效、生物皮肤均为 **Canvas / WebAudio 程序化原创生成**，不使用任何第三方美术素材。

在线试玩：部署后填入地址。

## 玩法

- **第一人称探索**：WASD 移动、空格跳跃、Shift 疾跑、鼠标控制视角（点击画面锁定指针）
- **体素世界**：噪声地形（平原 / 山丘 / 沙滩 / 湖海 / 树林）、3D 洞穴、硫磺洞穴群系，chunk 流式加载
- **挖掘与放置**：左键挖、右键放，射线选中高亮、分阶段裂纹、破坏粒子
- **生存模式**：生命 / 饥饿 / 氧气、摔落与溺水伤害、进食回复、死亡重生、掉落物拾取
- **工具与合成**：木/石/铁三阶 镐/斧/锹/剑，挖掘加速与掉落门槛、耐久度；E 键物品清单 + 合成面板（原木→木板→木棍→工具）
- **生物**：硫磺立方（可喂方块变弹球）、野猪、绒球羊、夜噬怪（夜间敌对、日光燃烧）、硫磺爆虫（逼近自爆）
- **昼夜循环**：太阳月亮升落、天色渐变、夜晚刷敌对生物
- **移动端支持**：手机 / iPad 自动显示虚拟摇杆 + 触屏按键，响应式 UI

进度自动保存到 localStorage（种子、方块修改、物品栏、玩家状态）。

## 本地运行

```bash
npm install
npm run dev      # 开发服务器 http://localhost:5173
npm run build    # 生产构建，产物在 dist/
npm run preview  # 预览构建产物
```

## 技术

- Vite + Three.js，纯前端无后端
- 每 chunk 合并几何体、仅渲染可见面、4 点顶点 AO
- 64×64 程序化写实贴图 + mipmap / 各向异性过滤

未使用任何 Minecraft 官方名称、Logo、贴图、音效、字体或 UI 图标。仅还原体素沙盒的通用玩法机制，全部资源为程序化生成。
